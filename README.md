# Aurelius

Bilingual EN/ES learning app — David's mascot to test the product before releasing it to the wild. Standalone organism, separated from the Hexelion dashboard.

## Scope Law (Important)

The EN/ES toggle lives here, in Aurelius. It does not belong to the Hexelion dashboard, whose language law is strict and immutable:

*   The Nexo → English
*   Le Jardin des Ombres → French
*   Internal doctrine → Spanish

Adding an EN/ES toggle to the Nexo or the Jardin would break this law. That is why Aurelius is its own repo: its bilingualism does not contaminate the two-skin organism.

## Status

Initial scaffolding. Only the i18n module is built (typed en/es translation object + toggle + subscription). The app itself (cards, quiz, progress) will be built on top when David decides.
